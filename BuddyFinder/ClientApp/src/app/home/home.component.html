Welcome to Buddy finder. Here you can find a teamate for the game you like the most. Feel free to create an account and wait for the generation of the teams. Our algorithm is always trying to generate balanced teams based on your age, weight, experience and number of matches. Also it is always learning once you give use the feedback after each match.

Create new player

This view is used to create a new player. Initially, the save button is disabled untill all the requirements are satisfied. Fill all the required fields and press the save button.

All players

This view is used to view all the existent players. You can also click on one of them to view the details of that player. Only a single player can be expanded at 1 point of time.

All teams

This view is used to view all the generated teams. You can also click on any team to view all the players inside it. It is possible to view more details at once.
